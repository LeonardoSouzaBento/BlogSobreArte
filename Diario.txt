A configuração flex: 1 1 auto; ou flex; 1 0 auto; (com aumento ativado) impede de ver o tamanho que for definido para a imagem (pelo aumento).

O width impede a imagem de crescer, mas não o min-width.

07/12/2024

O objet-fit (no, caso eu precisei do cover), só funciona se o elemento pai ter altura e largura especificadas.

08/12/2024

As divs filhas de uma div em display flex não crescem se o flex do pai não estiver coonfigurado. Ex.: flex: 1 0 240px;.

15/12/2024

As regras de overflow (como hidden) interferem na aplicação do display stick.
